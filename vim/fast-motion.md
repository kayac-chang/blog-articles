
### Forward
`f` + key

### Continue Forward / Backward
forward `;`
backward `,`

### Select current word
`viw`

### Select current word until space
`viW`

### Select inside brackets
`vi(`, `vi{`

### Select around brackets
`va(`, `va{`

### Yank around brackets
`ya(`, `ya{`

### Yank contiguous word
`yiw`, `yiW`
